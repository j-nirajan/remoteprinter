# Remote Printer — Basic Website

This is a small static site scaffold for the Remote Printer business. It includes:

- `index.html` — homepage with description and call-to-action
- `services.html` — list of typical printing services
- `about.html` — short business model explanation
- `contact.html` — contact details and instructions
- `styles.css` — minimal styling

Open `index.html` in a browser to view the site locally.
